Tired of accepting multiple salvage contracts and ending up with a pile of identical “Salvage Site” GPS markers?

This mod replaces the default salvage contract GPS clutter with normal, player-style GPS markers that are much easier to manage.

Instead of several markers all named:

**Salvage Site**

You get clean, readable markers such as:

**Salvage Site A**

**Salvage Site B**

**Salvage Site C**

The original contract marker is hidden, and a normal GPS marker is created at the same location. Since the new marker behaves like a regular GPS entry, it is much easier to rename, hide, remove, and organize.

## Features
- Converts salvage contract markers into normal GPS markers
- Automatically names them Salvage Site A, B, C, etc.
- Hides the original default Salvage Site marker
- Keeps the marker at the correct contract location
- Saves marker references between sessions
- Cleans up old replacement markers when the original contract marker is gone
- Works in singleplayer and multiplayer
- Dedicated server support
- Includes a reset command for restarting the salvage site letter sequence
  
## Why?

Vanilla salvage contract GPS markers are annoying because they all use the same name and cannot be properly renamed or removed like normal GPS markers.

This mod is a small quality-of-life fix for players who enjoy salvage contracts but want a cleaner GPS list and HUD.

## Important

This mod does not change the salvage contract itself.

It does not change:
- Contract rewards
- Contract objectives
- Salvage site locations
- Cargo contents
- Encounter generation

It only improves how the salvage contract GPS marker is handled.

## Recommended for
- Salvage contract players
- Multiplayer servers
- Players who like a clean GPS list
- Anyone tired of duplicate “Salvage Site” markers
